# Одностраничный сайт психолога

Откройте `index.html` в браузере.

Что заменить под реального специалиста:

- имя в заголовке и футере;
- Telegram-ссылку в блоке “Запись”;
- email;
- стоимость и формат консультаций;
- текст “О специалисте”;
- список тем, с которыми работает психолог.

Файлы проекта:

- `index.html` — страница сайта;
- `styles.css` — оформление;
- `assets/hero-office.png` — обложка.
